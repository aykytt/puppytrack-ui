# PuppyTrack UI

This repository contains UI components and design handoff documentation
for the PuppyTrack mobile app.

## Purpose
- Provide clear design decisions for developers
- Document component states and behaviors
- Share Figma references for implementation

## Handoff
Design handoff is done via:
- Figma file link (source of truth)
- Written component documentation in this repository
